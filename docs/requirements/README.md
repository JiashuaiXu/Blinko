## 🎯 需求概览

### 核心目标
开发一个 **桌面端（Windows）健康提醒工具**，主要功能：
- 通过摄像头或定时机制，提醒用户保持正确坐姿
- 鼓励用户多眨眼、定期休息，防止用眼疲劳
- 常驻系统托盘（Tray），轻量、不打扰但可交互

### 命名
- **中文名**：明眸
- **英文名**：Blinko

### 品牌理念
**内敛中式 + 外显科技萌** - 类似 Notion（极简） + 小爱同学（亲切）的结合
