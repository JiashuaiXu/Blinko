# 需求文档总结

本文档是对 Blinko / 明眸 项目需求的总结和索引。

---

## 📋 需求概览

### 核心目标
开发一个 **桌面端（Windows）健康提醒工具**，专注于保护用户的：
- 👁️ **眼睛健康**：通过眨眼检测和定时休息提醒
- 🪑 **坐姿健康**：通过摄像头监测坐姿，提醒保持正确姿势

### 产品定位
- **中文名**：明眸 - 文雅、文化感强、聚焦眼睛健康
- **英文名**：Blinko - 源自 "blink"（眨眼），轻快友好
- **品牌理念**：内敛中式 + 外显科技萌

## 🎯 关键需求点

### 功能需求
1. ✅ 定时提醒（20-20-20 法则）
2. ✅ 眨眼检测和提醒
3. ✅ 坐姿检测和提醒
4. ✅ 系统托盘常驻
5. ✅ 数据统计和可视化
6. ✅ 中英双语支持
7. ✅ 隐私优先（本地处理）

### 技术需求
1. ✅ Tauri + Rust + React + TypeScript
2. ✅ Python + OpenCV + MediaPipe（可选）
3. ✅ SQLite 数据存储
4. ✅ 本地 AI 处理

### 非功能需求
1. ✅ 启动时间 < 3 秒
2. ✅ 内存占用 < 100MB
3. ✅ 检测延迟 < 500ms
4. ✅ 崩溃率 < 0.1%
5. ✅ 隐私保护（本地处理）
