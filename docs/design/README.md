## 🎯 设计原则

### 核心原则：精简至上

- **精简优先**：最小化界面元素，只显示必要的信息
- **避免过度设计**：不预判未来需求，只实现当前需要的功能
- **轻量不打扰**：最小化界面侵入
- **友好且智能**：角色化设计增加亲和力
- **文化内核**：中文场景体现"明眸"文化底蕴
- **隐私透明**：清晰告知用户数据使用方式

**设计决策流程**：
1. 这个功能是否必需？
2. 能否用更简单的方式实现？
3. 能否删除某些部分？

**如果答案不确定，先不做。**
